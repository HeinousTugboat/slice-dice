## Usage

First, clone and run `npm install`.

Second, copy whatever images you want resized into `./input/`, ensuring that all files are images. NB: Any files smaller than `cropSize` will be enlarged and quality will be sacrificed.

Third, run `npm start`. It'll print out a list of each filename and its initial dimensions to console, putting each resized image into the `./output` directory.
